üìã Setup Instructions (One-time setup)

  1. Go to Google Sheets and create a new spreadsheet
  2. Name it something like "Weight Loss Tracker Data"
  3. In cell A1, type: groupName
  4. In cell A2, type: member0_name
  5. In cell A3, type: member0_weight
  6. Continue the pattern for members 1, 2, and 3 (cells A4-A9)
  7. Click Extensions > Apps Script
  8. Delete any code and paste the code from below
  9. Click the deploy button (top right) > New deployment
  10. Click the gear icon next to "Select type" and choose Web app
  11. Set "Execute as" to Me and "Who has access" to Anyone
  12. Click Deploy, authorize the app, and copy the Web app URL
  13. Paste the URL in the box below and click "Connect"
